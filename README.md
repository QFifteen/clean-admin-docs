## 在本地运行

1.克隆项目到本地
```bash
git clone https://github.com/QFifteen/clean-admin-docs.git
```

2.前往项目目录
```bash
cd clean-admin-docs
```

3.安装依赖(注意你的 Node 版本，这里默认你使用 pnpm)
```bash
pnpm i
```

4.启动本地服务
```bash
pnpm dev
```
